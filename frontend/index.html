Infraestrutura de Produção

Utilizamos tecnologias modernas como MySQL para o gerenciamento eficiente do banco de dados e Express.js no backend, proporcionando uma experiência ágil, estável e segura aos usuários.

Nossa plataforma foi desenvolvida com as melhores práticas, garantindo performance e confiabilidade.

Detalhes do Projeto

Nosso site permite que alunos e professores explorem TCCs de diferentes cursos, buscando por temas específicos e acessando os conteúdos de maneira prática.

O sistema conta com um mecanismo de aprovação para garantir a qualidade dos trabalhos publicados e ferramentas de busca avançada.

Quem Somos

Somos um grupo de estudantes do SENAI desenvolvendo um projeto inovador para organizar e exibir todos os TCCs já realizados na unidade.

Nosso objetivo é facilitar o acesso aos trabalhos acadêmicos, proporcionando uma plataforma intuitiva para alunos e professores.
